C语言中的&的用法

2022/8/28 其他 others C/C++

在做题时，经常会看到在数字或者字符串前面加上&，之前一直没太注意这个单目&，下面总结一下&的用法。😗

¶取地址（指针赋值）

个人认为c语言中，比较不容易理解的就是指针了。

举一个该情况下使用&的例子

#include<stdio.h>
int main()
{
    int a = 2;
    
    int*b;//定义一个整形指针
    
    b = &a;//给指针赋值，使指针指向a的地址
    
    printf("%d", b);//输出的是a的地址

    printf("%d", *b);//*的作用是解引用，取出指针指向地址的内容,本例里输出的是2
    return 0;
}

由于使用&（引用）的时候，跟原数指向的是一个地址，故也不会造成每次使用都新开辟一个内存，相当于JAVA中的静态变量

比如接下来在Tire中的一个insert操作（将相应的数字存储在Tire上，以便于接下来的操作）

void insert(int x){
    int p=0;
    for (int i = 30; ~i ; i -- ){
        int &s=son[p][x>>i&1];
        if(!s) s=++idx;
        p=s;
    }
}

这里在定义s时，如果不加第一个引用（第二个&是与操作，接下来会讲），只能代表son[p][x>>i&1]中存储的数值，而在这里使用了引用 就相当于以下操作

void insert(int x){
    int p=0;
    for (int i = 30; ~i ; i -- ){
        int &s=son[p][x>>i&1];
        if(!son[p][x>>i&1]) son[p][x>>i&1]=++idx;
        p=son[p][x>>i&1];
    }
}

是不是很省代码！

¶位操作中的与操作运算符

&是单目运算符，按位与运算通常用来对某些位清0或保留某些位

计算的时候按位计算，&两边操作数对应位上全为1时，结果的该位值为1。否则该位值为0

¶逻辑与运算

&&就是常说的and操作，双目运算符

逻辑运算结果只有0和1两种结果。 一般在&&两边是两个逻辑表达式。当两个表达式值均为1时，
&&运算结果为1，否则为0。
